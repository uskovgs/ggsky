# Celestial equator is the great circle with declination equal to 0 deg.

Celestial equator is the great circle with declination equal to 0 deg.

## Usage

``` r
equator
```

## Format

data.frame

- l:

  Galactic longitude, degrees.

- b:

  Galactic latitude, degrees.

- ra:

  Right ascension, degrees.
