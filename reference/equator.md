# Celestal equator is the great circle (decl=0deg)

Celestal equator is the great circle (decl=0deg)

## Usage

``` r
equator
```

## Format

data.frame

- l:

  Galactic longitude, degrees.

- b:

  Galactic latitude, degrees.

- ra:

  Right ascension, degrees.
